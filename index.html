MINOOTS

Distributed Horology Platform

Precision Timing

Distributed scheduling engine with microsecond accuracy. Schedule tasks across your infrastructure with guaranteed execution.

→ Cron-compatible scheduling
→ Interval-based timers
→ One-time delayed execution

Raft Consensus

Built on battle-tested Raft consensus algorithm for strong consistency. Automatic leader election and fault tolerance ensure reliability.

→ Automatic failover
→ Strong consistency guarantees
→ Split-brain protection

Webhooks & DLQ

HTTP webhooks with automatic retries and dead-letter queuing. Never lose a task execution with comprehensive failure handling.

→ Exponential backoff retries
→ Dead-letter queue for failures
→ Webhook signature validation

PostgreSQL Backed

Leverages PostgreSQL's ACID properties for durable timer storage. Your scheduled tasks survive crashes and network partitions.

→ ACID transaction guarantees
→ Efficient indexing for scale
→ SQL query interface

Getting Started

Quick Install

Deploy MINOOTS in minutes with Docker Compose or Kubernetes. Pre-configured for production with sensible defaults.

SDK Integration

Official SDKs for Go, Python, Node.js, and Rust. Simple APIs for scheduling, monitoring, and management.

Monitoring

Built-in Prometheus metrics and health check endpoints. Track execution rates, latencies, and cluster health.

Architecture

Horizontal Scaling

Add nodes dynamically to handle increased load. Automatic work distribution across the cluster.

Leader Election

Raft-based leader election ensures single authority. Automatic re-election on leader failure.

Heartbeat System

Continuous health monitoring with configurable intervals. Detect and recover from node failures automatically.

Timeline Queries

Query execution history and upcoming schedules. Audit trail for compliance and debugging.

Network Partitions

Resilient to network splits with Raft consensus. Maintains consistency even during disruptions.

Zero Downtime

Rolling updates without dropping scheduled tasks. Graceful shutdown with task handoff.
